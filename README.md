# Minecraft Server Install #

This is a simple ansible role that downloads and installs the Minecraft server jar into /opt/minecraft. 

```
minecraft_server_install
├── minecraft_install.yml
├── README.md
└── roles
    └── minecraft_server_install
        ├── files
        │   ├── eula.txt
        │   └── server.properties
        └── tasks
            └── main.yml
```
